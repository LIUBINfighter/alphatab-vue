# 吉他谱在线播放器

## 项目简介
这是一个基于 Vue.js 和 AlphaTab 库开发的在线吉他谱播放器应用。支持加载、查看和播放吉他谱，提供专业的乐谱显示和播放体验。

## 技术栈
- Vue 3 - 前端框架
- TypeScript - 类型支持
- Vite - 构建工具
- AlphaTab - 乐谱渲染引擎

## 主要功能

### 播放控制
- ⏯️ 播放/暂停/停止控制
- 🎮 播放进度实时显示
- 🎚️ 可调节播放速度（0.25x-2x）
- 🔄 循环播放功能

### 辅助功能
- 🎯 节拍器功能
- 🎼 预备拍功能
- 📐 多种布局模式
  - 垂直布局
  - 水平条形布局
  - 水平屏幕布局
- 🔍 乐谱缩放（25%-200%）
- 🖨️ 乐谱打印

## 特色
- 响应式设计，适配多种设备
- 支持 GP (Guitar Pro) 等主流吉他谱格式
- 专业级音色播放体验
- 实时进度跟踪

## 开发相关
- 使用 Vue 3 组合式 API 构建
- TypeScript 确保代码类型安全
- Vite 提供快速的开发体验

## 使用说明
1. 安装依赖：`npm install`
2. 开发模式：`npm run dev`
3. 构建项目：`npm run build`

## 问题日志

1. 远程加载字体不靠谱，记得将对应字体文件 `Bravura.otf` `Bravura.woff`复制到 `Public/fonts` 目录下。
